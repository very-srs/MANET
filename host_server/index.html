OpenTAKServer

TAK Server Mode:

(Consistency Mode)
Guarantees that no TAK data is ever lost. If the network splits, any smaller, isolated group will temporarily lose access to the TAK server until they rejoin the main group. Choose this for missions where data integrity is the top priority.
(Availability Mode)
Ensures a functional TAK server is always available to your local group, even during a network split.
WARNING: If your group rejoins the main network, all data created during isolation will be permanently lost. Choose this for missions where constant local communication is top priority.
